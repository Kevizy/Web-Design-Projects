Disadvantages of Ad Block

By: Kevin Izykowski


Although ad blockers can be useful, there are disadvantages. Here are some reasons why you should not be using ad blockers:


1. Websites could lose money

Whenever a visitor clicks on an ad, a website gets paid for that click. If a lot of users install ad blockers on their browsers, this could cause websites to lose profits. Eventually, a website could stop producing content or shut down.


2. Websites could not function properly

If a user has an adblocker installed on their browser, this could disable scripts. Some scripts are needed to make websites function properly.
